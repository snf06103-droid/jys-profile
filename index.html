Foundation & Standards

보이지 않는 곳의 견고함이 웹의 본질을 결정한다고 생각합니다.

화려한 시각적 효과도 결국 탄탄한 설계 위에서만 빛을 발합니다. 저는 화면의 겉모습을 복제하는 것에 그치지 않고,
웹 표준을 준수하는 시맨틱 마크업을 통해 정보의 위계가 명확하고 접근성이 보장된 '건강한 웹'을 구축합니다.

Core Philosophy

기본의 가치를 코드로 증명하는 저만의 고집입니다.

  • Semantic Markup

    단순히 div로 감싸는 것이 아니라, 검색 엔진과 스크린 리더가 콘텐츠를 정확히 해석할 수 있도록 적재적소에 태그를 사용합니다.

  • Responsive & Flexible

    다양한 디바이스 환경에 대응하는 반응형 웹을 구현합니다. CSS Grid와 Flexbox를 능숙하게 활용하여, 해상도가 변해도 레이아웃이 무너지지 않는 견고한 UI를 설계합니다.

  • Clean & Logic

    중복을 최소화하고 재사용이 용이한 CSS 구조를 지향합니다. 유지보수가 쉬운 코드가 곧 팀 전체의 비용을 줄이는 길임을 알기에, 네이밍 컨벤션과 코드의 가독성을 엄격하게 지킵니다.

Collaboration

팀의 생산성을 높이는 유연한 소통과 기록의 힘을 믿습니다.

  • Documentation

    누구나 이해할 수 있는 주석과 구조화된 폴더링을 통해 동료 개발자와의 협업 효율을 높입니다.

  • Git Workflow

    Main, Develop, Feature 등 명확한 브랜치 전략을 이해하고 있으며, 커밋 메시지 하나에도 작업의 의도를 담아 투명하게 공유합니다.

  • Problem Solving

    새로운 기술을 무작정 도입하기보다, 현재 프로젝트에 가장 적합하고 안정적인 기술인지 기초적인 원리부터 파악하고 적용합니다.

크리스마스 테마 팝업

크리스마스를 상징하는 딥 그린과 선명한 레드를 메인 컬러로 사용하면서도, 배경에 부드러운 아이보리 톤을 깔아 자극적이지 않고 따뜻한 연말의 감성을 표현했습니다.

일반적인 귀여운 크리스마스 컨셉보다는 무게감 있는 폰트와 질감이 느껴지는 오브제 이미지를 활용하여, 브랜드의 프리미엄한 가치를 강조했습니다.

뷰티 브랜드 팝업

'Small change, Remarkable difference'라는 슬로건에 맞춰, 깨끗하고 신뢰감 있는 코스메틱의 무드를 강조했습니다. 제품을 수직으로 쌓아 올린 구도를 통해 균형 잡힌 밸런스를 시각화했습니다.

무채색 배경에 그린 톤의 타이포그래피를 매치하여 내추럴하고 건강한 느낌을 주었으며, 버튼 디자인에 라운드 값을 주어 부드러운 이미지를 완성했습니다.

블랙 프라이데이 팝업

일반적인 블랙 프라이데이의 무겁고 강한 이미지 대신, 키치하고 트렌디한 핑크 컬러와 3D 느낌의 볼륨감 있는 텍스트를 사용하여 타겟층의 시선을 사로잡는 팝한 무드를 의도했습니다.

별빛이 흐르는 듯한 우주 배경과 핑크색 테이프 텍스처를 교차 배치하여 스타일리시한 공간감을 주었고, '최대 50%'라는 핵심 정보를 하단에 명확히 배치하여 클릭을 유도했습니다.

나이키 팝업

제품의 텍스처와 형태에 집중할 수 있도록 미니멀한 레이아웃을 구성했습니다. 거칠면서도 정돈된 배경 질감을 통해 에어포스 특유의 스트릿 감성과 클래식함을 동시에 표현했습니다.

대담하고 굵은 타이포그래피를 제품 뒤편에 배치하여 입체감을 주었으며, 전체적인 톤을 베이지와 브라운으로 통일하여 차분하면서도 힙한 느낌을 강조했습니다.

슈즈 브랜드 팝업

롱부츠의 우아한 라인을 강조하기 위해 대각선 구도를 활용하여 시각적인 역동성을 주었습니다. '서울' 컬렉션의 세련된 도시 감성을 담아내고자 했습니다.

다크 그린의 배경과 대비되는 옐로우 컬러의 타이포그래피로 주목도를 높였으며, 종이 질감이 느껴지는 배경을 사용하여 빈티지하면서도 고급스러운 프리미엄 무드를 연출했습니다.

크리스마스 팝업

정중앙에 배치된 트리와 어두운 배경을 통해 차분하고 고요한 크리스마스의 밤을 재현했습니다. 화려함보다는 정갈하고 따뜻한 '특별함'에 초점을 맞추었습니다.

'MERRY CHRISTMAS' 문구를 트리에 걸친 듯한 레이아웃으로 배치하여 시각적인 재미를 주었고, 하단 텍스트와 버튼에 얇고 깔끔한 폰트를 사용하여 여백의 미를 살린 고급스러운 팝업을 완성했습니다.

  • KIRIN

    팝아트적인 블루 톤과 과감한 수직 타이포그래피를 사용하여 청량함과 역동적인 브랜드 이미지를 강조했습니다.

    하프톤 패턴 배경과 과일 소스를 레이어드해 시각적인 재미와 함께 트렌디한 스트릿 감성을 담았습니다.

  • VELOLA

    차분한 아이보리 톤과 깊은 네이비 컬러를 매치해 프리미엄 카페의 고급스럽고 정갈한 이미지를 연출했습니다

    제품의 그림자를 살린 배치와 필기체 포인트가 조화를 이루어 감성적인 무드가 돋보이는 디자인입니다.

  • Daily Apple

    강렬한 레드 배경과 대비되는 화이트 타이포그래피로 '하루 한 알의 사과'라는 건강한 메시지를 직관적으로 전달합니다.

    클래식한 세리프 서체와 하단의 텍스트 배치를 통해 마치 잡지 광고 같은 세련된 레이아웃을 완성했습니다.

  • LAIT PUR

    웅장한 자연경관 배경 위에 산뜻한 옐로우 타이포를 얹어, 자연 유래 성분의 순수함과 생명력을 표현했습니다.

    광활한 풍경과 대비되는 제품의 클로즈업 배치를 통해 브랜드가 지향하는 깨끗하고 건강한 가치를 전달합니다
